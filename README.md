Overview

This week I made web pages interactive using JavaScript. I learned to:

- Select and traverse DOM elements
- Modify content, attributes, and styles
- Add, remove, and clone elements
- Handle events with listeners and delegation
- Validate forms and respond to user input

**Mini-Project: An interactive To-Do List where you can add, complete, delete, and filter tasks, with live stats.**

**Key Features**
- DOM Selection & Traversal: getElementById, querySelector, .children, .parentElement
- Content & Style Manipulation: textContent, innerHTML, dataset, .style
- Dynamic Elements: create, append, prepend, remove, clone
- Events: click, keyboard, form events, event delegation
- Form Handling: real-time validation and submission prevention
